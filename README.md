# hydra
A snake clone.

## How-to
  - Download
  - Run:

        pip install pygame
        python3 main.py
        
## Keys
  - Arrowkeys
  - [E] **↑**
  - [S] **←**
  - [D] **↓**
  - [F] **→** 

## Cheats
 - [A] automatically adds an apple to your diet and +1 snake length (levels increase according to rules)
 - [+] / [-] modifies framerate (and thus, speed)
